TIRA LAS BOTELLAS EN UN LUGAR CORRECTO

CONTAMINACIÓN POR BOTELLAS PET

Hoy se producen y se descartan en el país unas 200.000 toneladas anuales de envases PET, y si bien son perfectamente reciclables, solo se recuperan con este fin unas 70.000 toneladas (un 30%). Estas montañas literales de plástico generan, además del daño ambiental, una oportunidad de negocios desaprovechada. Es más, con la cantidad de botellas que se tiran diariamente se podría construir un edificio del tamaño de las torres Le Parc por día.

TERMOFORMDORA

El termoformado es un proceso de fabricación de plásticos que consiste en calentar una lámina de este material y darle formas específicas utilizando un molde. El plástico moldeado se enfría y se recorta para generar la pieza finalizada.

Termoformar e implementar techos verdes

Vamos a emplear un proceso de termoformado en las láminas de plástico PET para darles formas específicas mediante el uso de un molde, particularmente en este caso, adoptando la forma de macetas, con el propósito de crear un techo verde. Este sistema incluye vegetación y requiere un mantenimiento mínimo. Además, optimiza el aislamiento térmico, mejora la retención de calor en el edificio.

BOTELLA DE ACOPIO

Nuestra botella de acopio dispone de un programa de recolección de botellas PET que se integra con una base de datos enlazada a nuestra aplicación web. Este sistema tiene como objetivo proporcionar registros futuros sobre la capacidad de llenado y recolección de nuestras botellas cuando estén en funcionamiento en plazas públicas.
